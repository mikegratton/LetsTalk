Detailed Description

template<class Req, class Rep>
class lt::Requester< Req, Rep >

A lightweight requester wrapper.

A Requester is an object used for making requests to a remote service. Each request returns a future response that can be waited upon for the reply. These are constructed by the Participant.

Exceptions
std::runtime_errorif the service indicates an error.
