Detailed Description

template<class T>
class lt::ThreadSafeQueue< T >

A simple producer/subscriber queue using unique_ptr.

A thread-safe queue storing a list of unique_ptr<T>'s. Supports bulk push/pop operations. This queue is optionally bounded. If more items that the high water mark are inserted, the oldest items are discarded to keep the queue to the desired maximum size
